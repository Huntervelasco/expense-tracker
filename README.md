# expense-tracker

A simple CLI expense tracker made in Python.  
It allows you to register **expenses and incomes**, edit movements, filter by category, and calculate your balance.

## Features
- View all movements (expenses/incomes)
- Add expenses
- Add incomes
- Edit movements (type, category, amount, date, description)
- Filter movements by category
- Calculate total balance (income - expenses)
- Automatically saves data using JSON files

## Requirements
- Python 3.x
